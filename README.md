a spotify control display turned to do list display due to the discontinuiation of the free version of the api. this display utilises the todoist api, and displays tasks that you can control with the 3 buttons.
button 1: scrolls down the list
button 2: complete task
button 3: refresh list
the list can be updated from their website, which can update from your calendar (like google calendar or outlook, at least in theory)
